# omitKeys

Returns a modified version of the supplied object without the given keys.

## Installation

```bash
npm i boundless-utils-omit-keys --save
```

Then use it like:


```jsx
/** @jsx createElement */

import omitKeys from 'boundless-utils-omit-keys';

omitKeys({foo: 'bar', bar: 'baz'}, ['bar']); // returns `{foo: 'bar'}`
```
